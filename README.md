学习方法：
  1、看书、看文章：进度会比较缓慢，理解会相对困难一些。
  2、看教学视频：老师是非常有经验的开发者，其根据个人经验编排的课程，有条理性比较全面，对个人提升更直接更快。但免费的比较少，可能需要花费钱才能享受比较好的课程。
  3、源码学习：业界大佬写的开源框架代码学习是提升个人水平的一种非常有效的方式。如Spring、SpringBoot、SpringCloud、Netty等。
